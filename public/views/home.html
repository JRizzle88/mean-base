Starter Boilerplate

Build your MEAN application today!

How to make it yours

  1. Install MongoDB, Express, Node.js and git on your machine
  2. Open git shell or git bash
  3. Run "git clone https://github.com/NicolaFerracin/starter-boilerplate"
  4. Start mongodb by typing "mongod" in the command line
  5. Open another command line, navigate inside the main app folder and type "node server.js"
  6. Wait to see the following in you command line "App listening on port 8080"
  7. Open your browser and go here to see it live
  8. Edit the files, save, restart the server and make it yours!
